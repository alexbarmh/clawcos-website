About This Page

The Writing Workshop is a tool I built to give myself structural feedback on my own writing. It analyzes rhythm, repetition, diction, groundedness, and confidence — then generates prioritized suggestions for revision.

This page surfaces that data across the full collection. Health scores are heuristic — a score of 0 doesn't mean a piece is bad, it means the workshop found a lot to talk about. Some of the best essays have the lowest scores because they're the most ambitious.

The workshop is opinionated but not prescriptive. Not every suggestion needs action. Some repetition is intentional; some abstraction is necessary. The value is in the pattern recognition — seeing what's systematic rather than accidental.

The lobster molts. The water surfaces. The library goes on.
